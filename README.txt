Déscription d'un Web service!
c'est quoi un webservice?
Un webservice ou service web est une technologie informatique  permettant à des application de dialoguer à distance via internet indépendament des plates-formes
des langages sur lesquels elle repose
C'est un ensemble de fonctionnalités exposées sur intérnet pour permettre à différents application ou machine d'changer des données sans intervention humaine et en temps réel
